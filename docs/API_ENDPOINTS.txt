API BASE
- Base URL: /api/v1
- Content-Type default: application/json
- Auth endpoint protected: Authorization: Bearer <token>

RESPONSE FORMAT (umum)
- Success:
  {
    "success": true,
    "message": "OK",
    "data": ...,
    "meta": ... (opsional, untuk pagination)
  }
- Error:
  {
    "success": false,
    "message": "Pesan error",
    "errors": ... (opsional)
  }

!!! CATATAN PENTING (UPDATE) !!!
Semua path gambar sekarang adalah URL ABSOLUT (lengkap dengan http/https).
Flutter tidak perlu lagi menambahkan Base URL secara manual ke path gambar.
Gunakan langsung: Image.network(product.image)

CONTOH PENERAPAN DI FLUTTER
==================================================

1. MODEL PRODUCT (dart)
class Product {
  final int id;
  final String name;
  final String image; // URL Absolut dari API
  final double price;

  Product({required this.id, required this.name, required this.image, required this.price});

  factory Product.fromJson(Map<String, dynamic> json) {
    return Product(
      id: json['id'],
      name: json['name'],
      image: json['image'] ?? '', // Langsung pakai, sudah absolut
      price: (json['price'] as num).toDouble(),
    );
  }
}
